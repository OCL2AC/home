New Version of OCL2AC

To install the new version of our tool, please use the guidance described in the main webpage of our tool (https://ocl2ac.github.io/home/#installation) but be sure to use the following release URL instead of the one mentioned in the gaudiness (item 4).
The URL of the new release is: https://ocl2ac.github.io/home/sle/slerelease

We have implemented the following new optimizations in the new version of our tool OCL2AC:
(1)   Check-for-integration: A functionality which integrates a constraint only if it has reasonable overlap with the rule (as described in Section 5.3, Item 1 above). In the figure below we can see the wizard which provides not only the ability to integrate constraints into rules but also to check the need of the integration.





(2)   Simplification: A functionality which simplifies application conditions by applying equivalence rules (as described in Section 5.2). The figure below shows the corresponding functionally which should be invoked by the user (right click on a selected Henshin rule (opened by the tree-based editor) -> choose OCL2AC_Henshin -> Simplify Application Condition).
